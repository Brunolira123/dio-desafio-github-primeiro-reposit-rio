Comandos do Git primordiais:
ls - O comando do Bash ls é usado para listar o conteúdo do diretório de trabalho atual.  O ls é equivalente ao DIR em terminais de host do console do Window
git clone - Comando para clonar repositórios do GitHub para o local.
git status - Comando realizado para verificar o status do local para verificar se há ou não conteúdos para serem lançados para o commit
